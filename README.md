### Gulp

```gulp clean``` - очистить дирректорию dist
```gulp html```  - собирает html-файлы и копирует в дирректорию dist
```gulp styles``` - собирает scss-файлы, сжимает, переименовывает и копирует в дирректорию dist/css/
